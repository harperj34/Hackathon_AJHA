# flutter_application_1

A new Flutter project.

## How to open with link to database
1. Open two terminals
2. In terminal 1, cd flutter_application_1 then cd server, then "node index.js" to establish server connection. Should get a message saying "Connected to Neon database successfully!"
3. In terminal 2, run flutter run -d chrome as usual
